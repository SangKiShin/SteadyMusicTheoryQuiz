Steady Music Quiz는 직장인의 업무시간에 1시간마다 도착하는 퀴즈를 풀어보며, 틈틈이 음악 이론을 공부할 수 있는 텔레그램 채널입니다.

📅 전체 개요

🟢 사이클 1 (1~4개월)

  • C, F, G (3키)
  • Triad
  • Ionian, Dorian, Mixolydian

🟡 사이클 2 (5~8개월)

  • + D, Bb, Eb (6키)
  • + Open Triad
  • + Phrygian, Lydian, Aeolian, Locrian

🟠 사이클 3 (9~12개월)

  • + A, Ab, Db (9키)
  • + Drop 2
  • + Melodic Minor, Lydian b7, Altered

🔴 사이클 4 (13~16개월)

  • + E, B, Gb (12키)
  • + Drop 3
  • + Dorian b2, Lydian Aug, Mixo b6, Locrian #2

핵심 원칙

  • 사이클마다 3개 키 추가, 누적 (3→6→9→12)
  • 1~4주: 기존 보이싱 복습과 함께 새로운 키 점진적 도입
  • 5~8주: 새 보이싱 도입 + 새 스케일 점진적 추가
  • 9~16주: 복습, 심화, 통합
  • 9주: 음정 집중 주간 (스케일 모드 없음)
  • 보이싱 유형별 주당 1개 줄세트: 4/3/2주 단위 순환
  • 16개월 후 사이클 1부터 반복

음정 및 스케일 퀴즈

  • Find Note — 근음 + 음정 → 해당 음?
  • Find Interval — 근음 + 음 → 어떤 음정?
  • Mode Info — 카테고리 + 인덱스 → 모드 이름 및 공식
  • Mode Notes — 근음 + 모드 → 스케일 도수

보이싱 퀴즈

  • Triad — 3성부 기본 보이싱
  • Open Triad — 3성부 오픈 보이싱
  • Drop 2 — 4성부, 2번째 성부 1옥타브 하강
  • Drop 3 — 4성부, 3번째 성부 1옥타브 하강
💡 학습 팁학습 팁
  • 사이클마다 3개 키씩 추가. 3 → 6 → 9 → 12키.
  • 보이싱이 점진적으로 확장. Triad → Open Triad → Drop 2 → Drop 3.
  • 새 보이싱은 5주차부터 등장. 1~4주는 새 키 적응 기간.
  • 보이싱 유형별 주당 1개 줄세트. 한 포지션을 완전히 마스터하고 넘어가기.
  • 스케일도 같은 패턴. 핵심 3모드 → 메이저 7모드 전체 → 멜로딕 마이너 계열 → 완성.
  • 9주차는 음정 집중. 스케일 모드 없이 순수 음정 훈련.
  • 16개월 후 사이클 반복. 반복이 무의식적 숙달을 만든다.
  • 틀려도 괜찮다. 이건 테스트가 아니라 훈련. 매일 노출이 뇌를 바꾼다.
  • 보이싱 퀴즈는 기타로 연주. 생각만 하지 말고 손이 기억하게 하자.

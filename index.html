🎸 Steady Music Quiz

A 16-month master plan consisting of 4 cycles, each 4 months long.
▶️ Telegram 채널 — @SteadyMusicTheoryQuiz

📅 Overview

🟢 Cycle 1 (Months 1–4)

  • C, F, G (3 keys)
  • Triad
  • Ionian, Dorian, Mixolydian

🟡 Cycle 2 (Months 5–8)

  • + D, Bb, Eb (6 keys)
  • + Open Triad
  • + Phrygian, Lydian, Aeolian, Locrian

🟠 Cycle 3 (Months 9–12)

  • + A, Ab, Db (9 keys)
  • + Drop 2
  • + Melodic Minor, Lydian b7, Altered

🔴 Cycle 4 (Months 13–16)

  • + E, B, Gb (12 keys)
  • + Drop 3
  • + Dorian b2, Lydian Aug, Mixo b6, Locrian #2

Core Principles

  • 3 keys added per cycle, accumulating (3→6→9→12)
  • W01–W04: Gradually introduce new keys while reviewing existing voicings
  • W05–W08: New voicing introduced + new scales added progressively
  • W09–W16: Review, deepen, and consolidate
  • W09: Interval-focused week (no scale modes)
  • One string set per voicing type per week: Rotating on 4/3/2-week cycles
  • After 16 months, the cycle repeats from Cycle 1

🎯 Quiz Types

Interval & Scale Quizzes

  • Find Note — Root + interval → which note?
  • Find Interval — Root + note → what interval?
  • Mode Info — Category + index → mode name & formula
  • Mode Notes — Root + mode → scale degrees

Voicing Quizzes

  • Triad — 3-part basic voicing
  • Open Triad — 3-part open voicing
  • Drop 2 — 4-part, 2nd voice dropped octave
  • Drop 3 — 4-part, 3rd voice dropped octave

💡 Study Tips

  • Keys grow by 3 each cycle. 3 → 6 → 9 → 12 keys total.
  • Voicings open up progressively. Triad → Open Triad → Drop 2 → Drop 3.
  • New voicings appear from W05. W01–04 is an adaptation period for new keys.
  • One string set per voicing type per week. Master one position fully before moving on.
  • Scales follow the same pattern. Essential 3 modes → all 7 Major modes → Melodic Minor family → complete.
  • W09 is interval-focused. No scale modes — pure interval training.
  • The cycle repeats after 16 months. Repetition builds unconscious mastery.
  • Wrong answers are fine. This is training, not testing. Daily exposure rewires the brain.
  • Play voicing quizzes on your guitar. Don't just think — let your hands remember.
_A little every day, and in 16 months the foundations of jazz harmony will be second nature. 🎶_

🎶 Interval & Scale Quizzes

1. Find Note

Given a root and an interval, name the target note.

Q: What is the Major 3rd of C?
A: E
Q: What is the Minor 7th of G?
A: F

2. Find Interval

Given a root and a note, identify the interval between them.

Q: A is the ______ of F?
A: M3 (Major 3rd)
Q: Eb is the ______ of F?
A: m7 (Minor 7th)

3. Mode Info

Given a mode category and an index, name the mode and its interval formula.

Q: [Major Modes] What is the name and interval formula of mode 5th?
A: Mixolydian: R, 2, 3, 4, 5, 6, b7
Q: [Melodic Minor Modes] What is the name and interval formula of mode 7th?
A: Altered: R, b2, b3, b4, b5, b6, b7

4. Mode Notes

Given a root and a mode, spell out all the notes.

Q: Spell the notes of C Ionian.
A: C, D, E, F, G, A, B
Q: Spell the notes of D Dorian.
A: D, E, F, G, A, B, C

Voicing Types

  • Triad — 3-part close-position voicing (R, 3rd, 5th stacked tightly)
  • Open Triad — 3-part with wider spacing between voices
  • Drop 2 — 4-part voicing where the 2nd voice from the top is dropped down an octave
  • Drop 3 — 4-part voicing where the 3rd voice from the top is dropped down an octave
  • Drop 2&4 — 4-part voicing where both the 2nd and 4th voices are dropped

String Sets

Each voicing type uses specific string sets (1=e, 2=B, 3=G, 4=D, 5=A, 6=E). Quizzes are generated per string set, so you master one position at a time.
